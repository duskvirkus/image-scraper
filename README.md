# Image Scraper

Simple image scraper for the web. Built in support for tumblr.

## Usage

```bash
git clone https://github.com/violetcraze/tumblr-image-scraper.git
cd tumblr-image-scraper
npm i
npm start
```

Then follow instructions in electron app.
